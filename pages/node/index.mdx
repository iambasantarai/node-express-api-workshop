# What is Node.js ?

Node.js is a javascript runtime environment built on top of Chrome's v8 engine.
It allows us to develop application using javascript  outside of a web browser.
Some other jsvascript runtime environments are [deno](https://deno.com/), [bun](https://bun.sh/), [nappajs](https://github.com/microsoft/napajs) etc.

> Node.js is an asynchronous event-driven javascript runtime.

Node.js is versatile and well-suited for a variety of applications due to its asynchronous, event-driven architecture. 
Here are some practical use cases of Node.js:

- Web servers
- APIs
- Real-Time applications
- Microservices
- CLIs etc.
